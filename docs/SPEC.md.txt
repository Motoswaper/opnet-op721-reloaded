# OP721 Reloaded Specification

OP721 Reloaded is a **hash‑free, ERC‑721‑compatible NFT standard** for OP_NET.

## Core invariants

- `ownerOf(tokenId)` **MUST** return the current owner address.
- `balanceOf(owner)` **MUST** equal the number of tokens owned.
- Transfers **MUST**:
  - Clear approvals
  - Update balances
  - Update ownership
  - Emit `Transfer` events

## Required methods

- `balanceOf(owner: Address): u64`
- `ownerOf(tokenId: u64): Address`
- `transferFrom(from: Address, to: Address, tokenId: u64): void`
- `safeTransferFrom(...)` variants
- `approve(to: Address, tokenId: u64): void`
- `setApprovalForAll(operator: Address, approved: bool): void`
- `getApproved(tokenId: u64): Address | null`
- `isApprovedForAll(owner: Address, operator: Address): bool`

## Events

- `Transfer(from: Address, to: Address, tokenId: u64)`
- `Approval(owner: Address, approved: Address, tokenId: u64)`
- `ApprovalForAll(owner: Address, operator: Address, approved: bool)`
