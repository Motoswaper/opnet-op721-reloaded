# OP20 Reloaded — Fungible Token Standard for OP_NET

## Overview

OP20 Reloaded is the canonical fungible token standard for OP_NET.

It is analogous to ERC-20, but implemented in AssemblyScript for the OP_NET VM, with:

- deterministic behavior
- fixed storage layout
- no hidden minting or admin keys (by default)
- indexer- and DEX-friendly events

---

## Core interface

### Read methods

- `name(): string`
- `symbol(): string`
- `decimals(): u8`
- `totalSupply(): u64`
- `balanceOf(owner: Address): u64`
- `allowance(owner: Address, spender: Address): u64`

### Write methods

- `transfer(to: Address, amount: u64): bool`
- `approve(spender: Address, amount: u64): bool`
- `transferFrom(from: Address, to: Address, amount: u64): bool`
