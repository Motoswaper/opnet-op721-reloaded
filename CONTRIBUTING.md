# Contributing to OP721 Reloaded

Thank you for your interest in contributing!

## How to Contribute

1. Fork the repository  
2. Create a feature branch  
3. Commit your changes  
4. Submit a pull request  

## Coding Standards

- TypeScript only  
- Prettier formatting  
- ESLint rules enforced  
- All PRs must include tests  

## Reporting Issues

Open an issue with:

- Description  
- Steps to reproduce  
- Expected behavior  
- Actual behavior
